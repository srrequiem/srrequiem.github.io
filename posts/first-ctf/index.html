Intro#

El día 3 abril del 2021 TeamShakti publicó su anual competición CTF Beginner Friendly que lleva por nombre el mismo que el equipo ShaktiCTF. Tome la decisión de participar y probar suerte siendo que “oficialmente” nunca había participado en un CTF. Me pareció una buena oportunidad para practicar, aprender cosas nuevas y pulir mis habilidades, poniendo como objetivo (aunque lo considero apresurado) el alcanzar un rango aceptable en el próximo CTF de Hackthebox.

Presento un Write-up de los retos que pude resolver durante mi participación.

Delete - 100#

Descripción#

Sometimes what you see is not always true…

chall.png

Solución#

Se descargó la imagen y después analizar el archivo con binwalk se vio que había aparentemente dos imágenes en el mismo archivo, siguiendo la pista del título del reto se separaron los bytes iniciales que correspondían a la primera imagen usando dd

┌──(kali㉿kali)-[~/Desktop/shaktictf/forensics]
└─$ binwalk chall.png    

DECIMAL       HEXADECIMAL     DESCRIPTION
--------------------------------------------------------------------------------
0             0x0             PNG image, 264 x 191, 8-bit colormap, non-interlaced
48            0x30            PNG image, 1200 x 1200, 8-bit/color RGBA, non-interlaced
89            0x59            Zlib compressed data, default compression


┌──(kali㉿kali)-[~/Desktop/shaktictf/forensics]
└─$ dd skip=48 if=chall.png of=chall1.png bs=1                                
46287+0 records in
46287+0 records out
46287 bytes (46 kB, 45 KiB) copied, 0.284482 s, 163 kB/s

De esa manera chall1.png se vuelve una imagen que se puede visualizar con un visor de imágenes sin ningún problema. Obteniendo así la bandera.

Bandera: shaktictf{Y0u_4R3_aM4z1nG!!!!}

Crypto#

Classically Easy - 50#

Descripción#

ovmogeqfj{yahe_wgwff_avpv_eszahtmac_qxefowoey!}

Solución#

Realizando un poco de análisis a la cadena que se presenta en la descripción del reto se puede ver claramente que presenta un formato de bandera con excepción de los caractéres especiales, por lo que después de descartar las opciones que pudieran servir para descifrarla (Cifrado César, ROT N). Se trató de hacer otro alcance siguiendo el mismo principio de la rotación.

Char RecibidoValor de rotaciónChar Conocido
o+4s
v+12h
m+14a
o+22k
g+13t
e+4i
q+12c
f+14t
j+22f

Lo que podría significar un patrón númerico de rotación dada la repetición de los espacios recorridos del char dado y el char de la bandera, siendo +4,+12,+14,+22,+13.

Ejercicio manual de sustitución

shaktictf{lets_start_with_14,22,13,4,12,14,22,13,4_12,14,22,13,4,12,14,22,13!}

Optimización del método por medio de un script simple.

#!/usr/bin/python
abc = "abcdefghijklmnopqrstuvwxyz"
values = [4,12,14,22,13]
decrypt = "ovmogeqfj{yahe_wgwff_avpv_eszahtmac_qxefowoey!}"
avoid = "{}_!"
flag = ""
valIndex = 0
for letter in decrypt:
    if letter in avoid:
        flag += letter
        continue
    if valIndex >= len(values):
        valIndex = 0
    flag += abc[(abc.index(letter) + values[valIndex]) - len(abc)]
    valIndex += 1
print(flag)

Resultado del script

Bandera: shaktictf{lets_start_with_something_classical!}
