Solución#

Siguiendo las pistas y haciendo un poco de análisis del contenido del archivo que se nos presenta, pudieramos llegar a deducir que las cadenas presentadas son cadenas en hexadecimal y pudiera encontrarse la bandera en una de ellas de acuerdo con la descripción del reto, por lo que se hizo un pequeño script para intentar realizar fuerza bruta sobre el contenido (línea por línea) usando un byte como llave para el XOR.

#!/usr/bin/python3
from pwn import xor

f = open("output.txt", "r")
for line in f:
    for n in range(256):
          xorted = xor(bytes([n]), bytes.fromhex(line))
          try:
              decoded = xorted.decode("utf-8")
              if "CHTB" in decoded:
                  print(f"Flag: {xorted}")
          except:
              continue

Al ejecutarlo y esperar la respuesta (en mi caso fue por menos de 2 min) salió la bandera.

from Crypto.Cipher import AES
from Crypto.Util import Counter
import os

KEY = os.urandom(16)

def encrypt(plaintext):
    cipher = AES.new(KEY, AES.MODE_CTR, counter=Counter.new(128))
    ciphertext = cipher.encrypt(plaintext)
    return ciphertext.hex()


test = b"No right of private conversation was enumerated in the Constitution. I don't suppose it occurred to anyone at the time that it could be prevented."
print(encrypt(test))

with open('flag.txt', 'rb') as f:
    flag = f.read().strip()
print(encrypt(flag))

Solución#

Analizando el código y el output nos podemos dar cuenta que el output representa a los prints del script. La primera parte pertenece a la variable test del script, por lo que la segunda pertenece a la variable flag, ambas pasadas por la función encriptación. Prestando atención a la función de cifrado nos podemos dar cuenta que en la asignación de la variable KEY no la está recalculando para cada uso por lo que podemos deducir en la ejecución que se encargo de cifrar tanto la bandera como la cadena de prueba se utilizó la misma llave, siendo así una mala implementación del método de cifrado. Indagando acerca del funcionamiento del AES Counter Mode podemos ver que realiza un XOR por cada bloque del mensaje a cifrar como lo muestra la imagen:

Siendo:

  • Ek: Block cipher (AES) con k como llave.
  • m1, m2, … : Bloques del mensaje en texto plano
  • c1, c2, … : Bloques resultantes del mensaje cifrado
  • IV: Inicialización del vector usado por el counter

Quedando una ejemplificación de los datos que conocemos de esta manera:

* = Dato conocido
K = Llave usada
C1* = Primer texto cifrado
C2* = Segundo texto cifrado
P1* = Primer mensaje en texto plano
P2* = Segundo mensaje en texto plano

C1* = P1* xor K
C2* = P2 xor K (1)

Siguiendo las propiedades del xor podemos aplicar álgebra básica para despejar y obtener la información que desconocemos.

K = C1* xor P1* (2)

Sustituyendo (2) en (1) queda así y volvemos a despejar.

C2* = P2 xor C1* xor P1*

P2 = C2* xor C1* xor P1*

Llevando a la práctica lo anterior, podemos obtener la bandera del reto mediante el siguiente script:

#!/usr/bin/python3
from pwn import xor
# C2 xor C1
k = xor(bytes.fromhex("4b6f25623a2d3b3833a8405557e7e83257d360a054c2ea"), bytes.fromhex("464851522838603926f4422a4ca6d81b02f351b454e6f968a324fcc77da30cf979eec57c8675de3bb92f6c21730607066226780a8d4539fcf67f9f5589d150a6c7867140b5a63de2971dc209f480c270882194f288167ed910b64cf627ea6392456fa1b648afd0b239b59652baedc595d4f87634cf7ec4262f8c9581d7f56dc6f836cfe696518ce434ef4616431d4d1b361c"))

# C2 xor C1 xor P1
f = xor(k, b"No right of private conversation was enumerated in the Constitution. I don't suppose it occurred to anyone at the time that it could be prevented.")

print(f)
